查看文件的权限

  1. 项目根路径执行
  2. git ls-tree HEAD
  4. 结果
  5. 100644 blob 5551fde8e7dba1e37a1821e8b26374893e2a9e2e    mvnw （没有执行权限）

1.命令行添加执行权限

  1. git update-index --chmod=+x gradlew
  3. git commit -m "permission access for travis" （提交注释）
  5. 查询权限
  6. git ls-tree HEAD
  8. 100755 blob 5551fde8e7dba1e37a1821e8b26374893e2a9e2e    mvnw

2.直接在.travis.yml 中添加命令

  1. before_install:
  2.   - chmod +x mvnw
