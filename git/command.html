git命令

  1. 用法：git [--version] [--help] [-C <path>] [-c name=value]
  2.            [--exec-path[=<path>]] [--html-path] [--man-path] [--info-path]
  3.            [-p | --paginate | --no-pager] [--no-replace-objects] [--bare]
  4.            [--git-dir=<path>] [--work-tree=<path>] [--namespace=<name>]
  5.            <command> [<args>]
  7. 这些是各种场合常见的 Git 命令：
  9. 开始一个工作区（参见：git help tutorial）
  10.    clone      克隆一个仓库到一个新目录
  11.    init       创建一个空的 Git 仓库或重新初始化一个已存在的仓库
  13. 在当前变更上工作（参见：git help everyday）
  14.    add        添加文件内容至索引
  15.    mv         移动或重命名一个文件、目录或符号链接
  16.    reset      重置当前 HEAD 到指定状态
  17.    rm         从工作区和索引中删除文件
  19. 检查历史和状态（参见：git help revisions）
  20.    bisect     通过二分查找定位引入 bug 的提交
  21.    grep       输出和模式匹配的行
  22.    log        显示提交日志
  23.    show       显示各种类型的对象
  24.    status     显示工作区状态
  26. 扩展、标记和调校您的历史记录
  27.    branch     列出、创建或删除分支
  28.    checkout   切换分支或恢复工作区文件
  29.    commit     记录变更到仓库
  30.    diff       显示提交之间、提交和工作区之间等的差异
  31.    merge      合并两个或更多开发历史
  32.    rebase     在另一个分支上重新应用提交
  33.    tag        创建、列出、删除或校验一个 GPG 签名的标签对象
  35. 协同（参见：git help workflows）
  36.    fetch      从另外一个仓库下载对象和引用
  37.    pull       获取并整合另外的仓库或一个本地分支
  38.    push       更新远程引用和相关的对象
