Travis-CI 加密变量

linux安装

  1. yum install -y gcc ruby ruby-devel # 安装gcc和ruby环境
  3. gem sources --add https://gems.ruby-china.org/ --remove https://rubygems.org/ # 改为国内gem源
  5. gem install travis

登陆github账户

  1. travis login --pro
  3. 弹出框输入用户名/密码
  4. Username: 用户名
  5. Password for fartherp: 密码

加密

  1. travis encrypt -r owner/repo COVERALLS_TOKEN="secretvalue"
  3. owner：github用户名
  4. repo：github项目仓库名字

直接在.travis.yml 中添加命令

  1. env:
  2.   global:
  3.     - secure: ""
