数据库编码

查询数据库编码

  1. show variables like 'character%';
  2. show variables like 'collation%';

默认配置编码(配置文件my.cnf)

  1. [client]
  2. default-character-set=utf8
  3. [mysql]
  4. default-character-set=utf8
  5. [mysqld]
  6. character-set-server=utf8
  7. collation-server=utf8_general_ci

修改数据库级别的编码

  1. ALTER DATABASE `test` DEFAULT CHARACTER SET utf8 COLLATE utf8_general_ci;

修改表级别的编码

  1. ALTER TABLE `test` DEFAULT CHARACTER SET utf8 COLLATE utf8_general_ci;

修改表级别的编码

  1. ALTER TABLE `td_dic` CHANGE `dic_name` VARCHAR(45) CHARACTER SET utf8 COLLATE utf8_general_ci NOT NULL;
