函数错误

简介

  1. This function has none of DETERMINISTIC, NO SQL, or READS SQL DATA in its de 错误解决办法。

原因

  1. mysql开启bin-log时.
  2. 函数需要设置参数。
  3. 1: DETERMINISTIC 不确定的
  4. 2: NO SQL 没有SQl语句，当然也不会修改数据
  5. 3: READS SQL DATA 只是读取数据，当然也不会修改数据
  6. 4: MODIFIES SQL DATA 要修改数据
  7. 5: CONTAINS SQL 包含了SQL语句
  8. 函数只有1，2，3被支持。

设计

  1. mysql开启bin-log功能。

解决方法

  1. set global log_bin_trust_function_creators=TRUE;
