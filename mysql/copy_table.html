复制表

复制表结构及数据到新表

  1. CREATE TABLE td_dic_new SELECT * FROM td_dic;

只复制表结构到新表

  1. CREATE TABLE td_dic_new SELECT * FROM td_dic WHERE 1 = 2;
  2. CREATE TABLE td_dic_new LIKE td_dic;

复制旧表的数据到新表(两个表结构必须一致)

  1. INSERT INTO td_dic_new SELECT * FROM td_dic;

复制旧表的数据到新表(两个表结构不一致)

  1. INSERT INTO td_dic_new (id,dic_id) SELECT id,dic_id FROM td_dic;

可以将表1结构复制到表2

  1. SELECT * INTO td_dic_new FROM td_dic WHERE 1 = 2;

可以将表1内容全部复制到表2

  1. SELECT * INTO td_dic_new FROM td_dic;

列出表结构

  1. show create table td_dic;

mysqldump导出文件

  1. mysqldump -u username -p -d dbname tablename > tablename_new.sql;
